## Тема и стиль

### Концепция
**"Espresso & Rose"** — тёмный эспрессо-фон с розово-золотыми акцентами. Тепло, роскошь, Средиземноморье. Резко отличается от конкурентов (все используют белый/розовый/Elementor).

### Цвета
```css
--color-dark: #16100E     /* espresso dark — header, footer, hero */
--color-dark2: #1D1512    /* slightly lighter dark */
--color-bg: #F9F5F1       /* warm cream — light sections */
--color-light: #F2EAE2    /* blush warm — alternating sections */
--color-accent: #C9917A   /* terracotta rose gold */
--color-gold: #B5956A     /* classic gold — secondary accent */
--color-text: #2A1F1C     /* dark brown text */
--color-text-light: #EDE5DC /* warm white for dark bg */
--color-border: #E6D9CF
```

### Шрифты
- **Cormorant Garamond** — заголовки (элегантность, французский/испанский шик)
- **Lato** — тело (читаемость)
